

Automation using Java, BDD

It includes the following :

Project Structure:

1. Basic framework with few features files
2. Feature file is added in \Features folder
3. Test Runner and step definition is added in src\main\java
4. Properties are added in Configuration.Properties file
5. Following are the dependencies added in Pom.xml:
	cucumber-core
    cucumber-java
	cucumber-junit
	cucumber-jvm-deps
	cucumber-reporting
	gherkin
	junit
	

Highlights of the Project:

1. It is maven project using Cucumber and Selenium libraries.
2. Configuration properties are added in Configuration.Properties file
3. Using CucumberOptions in Runner file, html report are generated at target/CucumberHtmlReport.html.
